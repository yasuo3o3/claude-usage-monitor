# Claude Usage Monitor

Claude の使用量をリアルタイムで確認できる Chrome 拡張機能です。

> ⚠️ **非公式ツールです。** Anthropic 社とは一切関係ありません。
>
> This is an unofficial tool and is not affiliated with Anthropic.

## 機能

- 5時間セッションの使用量をプログレスバーで表示
- 週間制限（全体 / Sonnet / Opus）の使用量を表示
- リセットまでの残り時間を表示
- 使用量が高くなると警告メッセージを表示
- 拡張機能アイコンにバッジで使用率を表示（50%超で黄、80%超で赤）
- 5分ごとに自動更新

## インストール方法

1. このリポジトリをダウンロード（ZIP）または `git clone`
2. Chrome で `chrome://extensions/` を開く
3. 右上の **「デベロッパーモード」** を ON にする
4. **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. ダウンロードしたフォルダを選択

## 使い方

1. [claude.ai](https://claude.ai) にログインした状態で使用してください
2. Chrome ツールバーの拡張機能アイコンをクリックすると使用量が表示されます
3. 🔄 ボタンで手動更新もできます

## 注意事項

- claude.ai にログインしている必要があります
- claude.ai の内部 API を利用しているため、サイトの仕様変更により動作しなくなる場合があります
- 取得したデータはブラウザのローカルストレージにのみ保存され、外部には一切送信されません
