# Claude Usage Monitor

Claude の使用量をリアルタイムで確認できる Chrome 拡張機能です。

> ⚠️ **非公式ツールです。** Anthropic 社とは一切関係ありません。
>
> This is an unofficial tool and is not affiliated with Anthropic.

## 機能

- 5時間セッションの使用量をプログレスバーで表示
- 週間制限（全体 / Sonnet / Opus）の使用量を表示
- リセットまでの残り時間を表示
- バックグラウンドで5分ごとにデータを自動取得・更新

## 🎨 色と表示のルール

本拡張機能では、現在の「実際の使用量」が「時間経過のペース（期待値）」に対してどうなっているかを計算し、色で直感的に教えてくれます。

### 📊 プログレスバー（ポップアップ画面）
*   **緑色（通常ペース）**: 時間経過に対して余裕がある状態。そのまま使って大丈夫です。
*   **黄色（やや早いペース）**: プログレスバー自体も黄色になり、背景に「本来はこのぐらいのはず」という期待値バー（薄い黄色）が表示されます。少しペースを落とした方が良いサインです。
*   **赤色（オーバーペース・制限到達）**: ペースを大幅に超過しているか、制限に到達（100%）した状態です。期待値バーは非表示になります。

### 🏷️ アイコンバッジ（拡張機能アイコン右下の数字）
アイコンには `H25`（5時間で25%使用）や `W40`（1週間で40%使用）のように、15秒ごとに交互に表示されます。

*   **緑色**: 基本的にバッジは表示されません（気にする必要がないため）。
*   **黄色**: ペースが早まっている時に表示されます。少し注意が必要です。
*   **赤色**: 上限が近い（または超えている）危険な状態です。早急にペースを落とすか、無料枠から切り替える必要があります。

## インストール方法

### 1. ダウンロード

**方法A: ZIPでダウンロード（推奨）**
1. このページ上部の緑色の **「Code」** ボタンをクリック
2. **「Download ZIP」** を選択
3. ダウンロードした ZIP ファイルを解凍

**方法B: Git を使う場合**
```bash
git clone https://github.com/yasuo3o3/claude-usage-monitor.git
```

### 2. Chrome に拡張機能をインストール

1. Chrome で `chrome://extensions/` を開く
2. 右上の **「デベロッパーモード」** を ON にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. 解凍したフォルダ（`claude-usage-monitor` フォルダ）を選択

## 使い方

1. [claude.ai](https://claude.ai) にログインした状態で使用してください
2. Chrome ツールバーの拡張機能アイコンをクリックすると使用量が表示されます
3. 🔄 ボタンで手動更新もできます

## 注意事項

- claude.ai にログインしている必要があります
- claude.ai の内部 API を利用しているため、サイトの仕様変更により動作しなくなる場合があります
- 取得したデータはブラウザのローカルストレージにのみ保存され、外部には一切送信されません
